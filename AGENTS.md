# textus

> A coordination space for humans, AI, and automation ‚Äî durable, multi-writer repo memory that survives the session, the model, and the vendor.

## Developer Commands
- **test**: `bundle exec rspec`
- **lint**: `bundle exec rubocop`
- **lint-fix**: `bundle exec rubocop -A`
- **setup**: `bundle install`
- **help**: `bundle exec exe/textus --help`
- **drain**: `bundle exec exe/textus drain --as=automation`
